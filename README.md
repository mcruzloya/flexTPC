# A flexible model for thermal performance curves

This repository contains the data and code for the manuscript Cruz-Loya M, Mordecai EA, Savage VM 2024. A flexible model for thermal performance curves. \[[Preprint available in biorXiv](https://doi.org/10.1101/2024.08.01.605695)\]

Data and code for the model comparison in the insect development, glacier bacteria and antibiotic TPCs datasets (Figure 2 in the manuscript) is provided under the `model_comparison` folder.

Data and code for fitting the mosquito traits (Figure 3 in the manuscript) is provided under the `mosquito_traits` folder.
